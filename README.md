# Дизельный двигатель для грузового автомобиля 4ЧН 10.5/12.8 — Выпускная квалификационная работа

Тема ВКР: Дизельный двигатель для грузового автомобиля 4ЧН 10.5/12.8 мощностью Ne = 140 кВт на номинальной частоте n = 2300 об/мин

## В этом репозитории содержится исходный код, вычислительные данные и результаты моделирования, написанные в рамках моего дипломного проекта. В проекте в качестве прототипа используется ЯМЗ-5340, а его основные задачи включают в себя:
1、Результаты оптимизации рабочего процесса на основе программного обеспечения Diesel-RK.

2、Конечно-элементное моделирование стационарного температурного поля поршня (двумерная осесимметричная модель).

3、Моделирование сопряженного температурного поля поршневого кольца с учетом динамического контакта и теплового равновесия.
## 📁 Часть 1: Результаты оптимизации рабочего процесса Diesel-RK
В этом разделе представлены параметры дизельного двигателя при различных условиях эксплуатации, полученные путем многопараметрического сканирования и оптимизации с использованием программного обеспечения Diesel-RK.
## 📁 Часть 2: Программа конечно-элементного анализа температурного поля поршня
Метод конечных элементов для решения двумерных осесимметричных задач стационарной теплопроводности с использованием Python.
### Основные функции
Импортируйте файл сетки ANSYS ICEM в формате .msh

Постройте глобальную матрицу теплопроводности для осесимметричных треугольных элементов

Примените три типа граничных условий (тип I, тип II, тип III)

Решите систему линейных алгебраических уравнений
### Выход
1、Карта распределения температурного поля поршня

2、Карта распределения температурного поля поршня после итеративной оптимизации

3、Карта распределения разницы температур до и после оптимизации
## 📁 Часть 3: Программа моделирования сопряженного температурного поля поршнево-кольцевого узла
Данная программа рассчитывает давление газа между кольцами, осевую силу поршневого кольца, теплопередачу в канавках кольца и масляную пленку смазки (уравнения Рейнольдса) для оптимизации входных значений граничных условий третьего типа для температурного поля поршня.
### Основные функции
Расчет рабочего цикла двигателя (0–720°CA) (включая продувку, выделение тепла и рассеивание тепла)

Расчет межкольцевого давления (модель дросселирующего объема)

Анализ кольцевых осевых сил (сила газа, сила инерции, сила трения)

Определение состояния кольцевого контакта (прижим верхней торцевой поверхности/прижим нижней торцевой поверхности)

Расчет коэффициента теплопередачи на поверхности кольцевой канавки

Решение уравнения Рейнольдса (толщина масляной пленки между кольцом и гильзой цилиндра)
### Входной файл（Входные данные результатов моделирования для сравнения и отображения.）
1、Давление в цилиндре изменяется в зависимости от угла поворота коленчатого вала (выходной сигнал Diesel-RK).
2、Температура в цилиндре изменяется в зависимости от угла поворота коленчатого вала (выходной сигнал Diesel-RK).
### Выход
1、Кривые давления в цилиндре и задней части кольца
2、Кривые осевой силы кольца в зависимости от трения
3、Переворот кольца (состояние плотного контакта) в канавке кольца
4、Кривая коэффициента теплопередачи на поверхности канавки кольца
5、Кривые зависимости толщины масляной пленки от коэффициента теплопередачи между кольцом и гильзой цилиндра
